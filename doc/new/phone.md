## 手机端今日营业额

**url:** /phone/todaydata

**HTTP请求方式:** GET

**请求参数:**

参数名称|类型|描述
---|:--:|---:
shopid|int|如果是商店登录查看本店经营分析表，不需要此项，如果是brand查看shop的经营分析，需要传入shopid

**返回格式:** json


**返回内容:**
```

{
    "msg": "成功",
    "code": 0,
    "data": {
        "项目数": 3,
        "人头": 2,
        "客人单价": 96.33,
        "客流": 3,
        "实操数": 3,
        "现金": 234.0
    }
}

```
