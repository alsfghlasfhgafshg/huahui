## 获取所有支付方式id

**url:** /settlement/allpaymentmethod

**HTTP请求方式:** GET

**请求参数:** 无参数

**返回格式:** json

**返回内容:**
```

返回array
id：支付方式id
name：支付方式名称
{
    "msg": "成功",
    "code": 0,
    "data": [
        {
            "id": 1,
            "name": "XXX"
        }
    ]
}

```
